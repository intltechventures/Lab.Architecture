## Open Source Tools

- Apache Airflow®
  + Used to programmatically author, schedule, and monitor workflows as Directed Acyclic Graphs (DAGs) of tasks using Python. It excels at managing complex ETL/ELT pipelines, offering dynamic pipeline generation, robust monitoring via a web UI, and scalable task execution.
  + https://airflow.apache.org/
  + https://github.com/apache/airflow
  + https://en.wikipedia.org/wiki/Apache_Airflow
